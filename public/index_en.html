taruntarun relay

It is a relay server for Mastodon instance managed by @mayaeh
It is supposed to be small scale use mainly for small instances for Japanese speakers.

Getting Started

Open Mastodon WebUI, move to Settings -> Administration -> Relays.
Please press the "ADD NEW RELAY" button, copy & paste the following URL into the "URL of the relay inbox".

https://relay.taruntarun.net/inbox

Please read the terms of service before registering.
After registering and starting to use, we assume that you have agreed to the terms of service.

Terms of Service

This relay server is mainly intended for small instance connections for Japanese speakers.

Please do not connect instances where the following acts are done.

  • Action contrary to public order and morals
  • Acts that conflict with laws in Japan
  • Acts that hinting at crime
  • Acts that defaming or damaging others
  • Acts that for collecting information
  • Acts that to place load on this server (such as going offline frequently)
  • Other acts of putting excessive burden on the server

Please use "Unlisted" for posting by BOT whenever possible.

Please cancel the registration before stopping the instance for a long time.

If the above is not followed, registration may be suspended unilaterally.
